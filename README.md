<h1 align="center">Random Users</h1>

### ✨ Requirements
* **OS:** Android 10 or higher
* **SDK:** API Level 29+

### 🛠 Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** Clean Architecture + MVI
- **Concurrency:** Kotlin Coroutines, Flow
- **Dependency Injection:** Dagger Hilt
- **Storage**: Room
- **Network**: Retrofit, Coil
- **Pagination**: Jetpack Paging 3

### 📥 Build App

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kolisnichenko2828/Random-Users.git
   ```
2. **Open the project in Android Studio.**
3. **Add API Key:**
   * Get your key from [Ninjas API](https://api-ninjas.com/api/randomuser).
   * Add it to your local.properties file:
   ```properties
   RANDOM_API=YOUR_API_KEY
   ```
4. **Build the app or run it on device.**
